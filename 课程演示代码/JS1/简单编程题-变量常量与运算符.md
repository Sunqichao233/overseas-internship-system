# 题目：奶茶店营业额计算器（变量、常量、运算符）

请用 JavaScript 完成下面功能：

1. 使用变量保存：
- 奶茶单价（元）
- 今日售出杯数
- 满减金额（元）

2. 使用常量保存：
- 税率 `6%`（`TAX_RATE`）

3. 计算并输出：
- 原价总额 = 单价 × 杯数
- 应缴税额 = 原价总额 × 税率
- 实收金额 = 原价总额 + 应缴税额 - 满减金额

4. 使用 `console.log` 输出三项金额（保留 2 位小数）。

可用测试数据：
- 单价：18
- 杯数：35
- 满减：20

（涉及知识点：`let`、`const`、`+ - *`、括号运算）
