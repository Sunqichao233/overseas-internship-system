题目3：报名页（表格+表单）

要求：不使用 CSS，只用 HTML 标签实现“课程信息 + 报名表单”。

任务A：课程时间表

  1. 做一个 4 列表格：日期、时间、课程主题、讲师。
  2. 使用 caption、thead、tbody、tfoot。
  3. 至少填写 3 行课程数据。

任务B：报名表单

  1. 表单中包含：姓名、学号、邮箱、手机、报名课程（下拉框）、学习基础（单选）、可上课时间（复选）、备注（文本域）。
  2. 输入项要配套 label，并与对应控件关联。
  3. 使用 fieldset 与 legend 分组。
  4. 包含提交按钮和重置按钮。

标签要求

至少使用：table、tr/th/td、form、input、select/option、textarea、button。
